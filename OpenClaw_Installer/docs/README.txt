================================================================================
                        OpenClaw 自动部署包
                      AI Gateway for Chat Apps
================================================================================

【项目简介】

OpenClaw 是一个自托管的 AI Gateway，连接您的聊天应用（WhatsApp、Telegram、
Discord、iMessage 等）到 AI 编码助手。您在自己的机器上运行 Gateway，它
将成为消息应用与 AI 助手之间的桥梁。

================================================================================

【系统要求】

- 操作系统: Windows 10/11 (64位)
- Node.js: 22+ (必需)
- 内存: 4 GB RAM
- 网络: 稳定的外网连接

================================================================================

【包含内容】

1. 安装脚本 (scripts/)
   - install.bat - 批处理安装脚本 (推荐)
   - install.ps1 - PowerShell 安装脚本 (支持更多选项)
   - uninstall.bat - 卸载脚本

2. 可选: Node.js 安装包
   - 请从 https://nodejs.org/ 下载

================================================================================

【快速开始】

方法一：批处理脚本 (推荐)
-------------------------
1. 确保已安装 Node.js 22+
2. 以管理员身份运行 scripts\install.bat
3. 按照提示完成安装
4. 编辑配置文件启用消息通道
5. 启动 OpenClaw

方法二：PowerShell
------------------
1. 确保已安装 Node.js 22+
2. 以管理员身份运行:
   .\install.ps1 -InstallPath "C:\OpenClaw"

方法三：手动安装
----------------
1. 安装 Node.js 22+
2. 创建安装目录
3. 运行: npm install -g openclaw
4. 创建配置文件: ~/.openclaw/openclaw.json
5. 运行: npx openclaw

================================================================================

【配置文件】

配置文件位置: %USERPROFILE%\.openclaw\openclaw.json

默认配置示例:

{
  "channels": {
    "whatsapp": {
      "enabled": false
    },
    "telegram": {
      "enabled": false
    },
    "discord": {
      "enabled": false
    }
  },
  "server": {
    "port": 18789,
    "host": "127.0.0.1"
  }
}

启用 WhatsApp 示例:
{
  "channels": {
    "whatsapp": {
      "enabled": true,
      "allowFrom": ["+15555550123"]
    }
  }
}

================================================================================

【控制面板】

安装完成后，访问: http://127.0.0.1:18789/

================================================================================

【故障排除】

1. 提示 "node" 不是内部或外部命令
   - 确认 Node.js 已正确安装
   - 重启终端或电脑

2. 端口 18789 被占用
   - 修改配置文件中的端口号
   - 或关闭占用该端口的程序

3. 消息通道无法连接
   - 检查网络连接
   - 确认 API 密钥配置正确
   - 查看日志: %USERPROFILE%\.openclaw\logs\

================================================================================

【卸载】

方法一: 运行 uninstall.bat
方法二: 通过 Windows 控制面板 -> 程序和功能 -> 卸载 OpenClaw
方法三: npm uninstall -g openclaw
