飞书配置指南

将 OpenClaw 接入飞书，打造专属 AI 助手

FEI

飞书 / Lark

https://open.feishu.cn/app
  • 企业级即时通讯平台
  • 支持机器人对话
  • 支持群聊和私聊
  • 支持代码块导入和执行
  • 免费使用

步骤一：创建飞书应用

  1. 访问 飞书开放平台 并登录
  2. 点击「创建应用」→ 选择「企业自建应用」
  3. 填写应用名称（如：AI助手）和描述
  4. 进入应用 → 左侧菜单「凭证管理」→ 复制 App ID 和 App Secret

步骤二：配置机器人权限（重要）

进入「权限管理」页面，搜索并开通以下权限：

权限名称 权限ID 用途
获取与发送消息 im:message 基础消息收发
获取群组信息 im:chat 群聊功能支持
以应用身份读取群消息 im:chat:readonly_as_app 读取群聊消息
接收群聊消息 im:message.group_msg 接收群消息事件
以应用身份发消息 im:message:send_as_bot 机器人发送消息
上传文件到云文档 drive:file:upload 上传代码文件
下载云文档文件 drive:file:download 下载代码文件
查看云文档 docs:doc:readonly 读取文档内容
编辑云文档 docs:doc 编辑文档/代码
获取用户信息 contact:user.base:readonly 获取用户基本信息

权限配置提示

权限配置完成后，需要发布新版本才能生效。建议一次性配置所有权限后再发布。

步骤三：配置事件订阅（重要）

进入「事件订阅」页面，添加以下事件：

事件名称 事件类型
接收消息 im.message.receive_v1
群组被创建 im.chat.created_v1
群组信息变更 im.chat.updated_v1
用户进入群组 im.chat.member.added_v1
文件创建 drive.file.created_v1
文件更新 drive.file.updated_v1

步骤四：配置机器人

  1. 左侧菜单「机器人」→「添加机器人」
  2. 填写机器人名称和头像
  3. 配置机器人描述和简介
  4. 设置「命令」支持（可选）：
    • /help - 获取帮助
    • /code - 代码模式
    • /clear - 清除对话

步骤五：配置 OpenClaw

安装飞书插件 
npx openclaw configure feishu

按照提示输入：

  1. Select channel → 选择 feishu/lark
  2. Install feishu plugin? → 选择 Download from npm (@openclaw/feishu)
  3. App ID → 输入飞书应用的 App ID
  4. App Secret → 输入飞书应用的 App Secret
  5. Feishu domain → 选择 feishu.cn - China
  6. Group chat policy → 选择 open

步骤六：配置长连接

  1. 启动 OpenClaw Gateway：npx openclaw gateway
  2. 返回飞书开放平台 → 左侧菜单「事件订阅」
  3. 选择「长连接」模式（推荐）或「Webhook」模式
  4. 长连接无需配置服务器，适合本地开发
  5. Webhook 需要配置公网可访问的服务器地址

步骤七：发布应用

  1. 左侧菜单「版本管理与发布」→「创建版本」
  2. 填写版本号（如：1.0.0）和版本说明
  3. 点击「申请发布」
  4. 企业自建应用通常免审核，直接发布
  5. 发布后，在飞书中搜索机器人名称即可使用

步骤八：用户绑定

批准用户绑定 
npx openclaw pairing approve feishu XXXXXX

用户首次在飞书中给机器人发消息时，会收到绑定码提示。管理员需要在命令行执行上述命令完成绑定。

代码导入功能

OpenClaw 支持从飞书云文档导入代码文件：

  • 支持格式：.py, .js, .ts, .java, .go, .rs, .cpp, .c, .h 等
  • 在飞书中上传代码文件到云文档
  • 分享文件链接给机器人
  • 机器人可读取并分析代码内容
配置代码导入权限 
npx openclaw config set plugins.feishu.codeImport true
npx openclaw config set agents.defaults.permissions.allowReadFiles true
npx openclaw config set agents.defaults.permissions.allowWriteFiles true

提示

OpenClaw 2026.2.6 及以上版本已内置飞书插件，安装时可选择 "Skip for now" 跳过下载。代码导入功能需要开通云文档相关权限。

{

}

常见问题

常见问题解答

API Key 安全性如何？

API Key 只存储在本地配置文件中，不会上传到任何服务器。建议定期更换密钥，避免泄露。

如何查看 API 调用费用？

各平台都提供控制台查看使用量和费用，建议定期检查消费情况，设置预算提醒。

可以同时使用多个 Provider 吗？

可以！您可以配置多个 provider，通过修改 agents.defaults.model 在不同场景下切换使用。

推荐哪个平台？

新手推荐： 硅基流动（性价比高，注册就送大量额度）
企业用户： 阿里云百炼（稳定可靠，企业级支持）
